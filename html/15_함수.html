함수(Function, 기능)

    - 목표하는 결과를 만들기 위한 코드의 묶음

    - 코드를 {} (블록)으로 묶어서 작성하고
     붙여진 이름을 호출해서 사용


    [함수 기본 작성법]
    // []  : 생략 가능(안써도 됨)
    // ... : n개 작성 가능(여러 개)
    
    function 함수명([매개변수...])  // 함수 선언
    {
      // 함수 정의
    }

    함수명([전달인자...]); // 함수 호출

전달 인자(Argument), 매개 변수(Parameter)

    * 전달 인자(Argument)
      - 함수 "호출" 시 () 안에 작성되는 값
      - 작성된 값은 함수의 매개 변수로 전달됨

    * 매개 변수(Parameter)
      - 함수 "선언" 시 () 안에 작성되는 변수
      - "호출" 시 전달되는 값(전달 인자)를 저장하기 위한 변수
      - 매개 변수는 함수의 지역 변수(함수에서만 사용 가능)

입력 받은 수 3개의 합 구하기

입력 1 :
입력 2 :
입력 3 :

입력 받은 값 A가 B의 배수가 맞는지 확인

A :
B :

return (반환)

    - 함수 내에 작성하는 명령어

    - 현재 수행 중인 함수를 종료하고
      호출한 곳으로 돌아가라!

    - return;    -> 함수 종료, 호출한 곳으로 돌아감 

    - return 값; -> 함수 종료, 호출한 곳으로 값을 가지고 돌아감 

    (참고)
    함수 정의 시 return을 작성하지 않은 경우
    자동으로 함수 정의 끝 부분에 return 구문이 추가된다.
