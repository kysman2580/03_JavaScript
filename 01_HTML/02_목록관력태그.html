li (List Item) 태그 : 목록의 한 줄

  • 김치
    • 갓김치
    • 열무김치
    • 총각김치
  • 죽

    • 김치죽
    • 낙지죽
  • 라면
    • 김치라면
    • 떡라면

  • 국밥

    • 특국밥

ol(ordered List) 태그 : 순서가 있는 목록(숫자가 붙음)

  1. 우육면
  2. 김밥
  3. 쉑쉑버거
        * type 속성 : 표시되는 숫자 형식 지정

        type="a" : 영어 소문자(a,b,c,d...)
        type="A" : 영어 대문자(A,B,C,D...)
        type="i" : 로마 소문자(i,ii,iii,iv,v,vi...)
        type="I" : 로마 대문자(I,II,III,...)
        type="1" : 숫자(1,2,3...) (기본값 = 아무것도 적지 않았을 때 적용)
*start 속성 : 목록 시작 번호 *reversed : 순서 반대로(역순)
  1. HTML5
  2. CSS3
  3. JavaScript
  4. Java
  5. DB(Oracle)

Emmet : HTML, CSS 빠른 자동완성 지원 플러그인

      태그명 : 태그 시작/종료 자동 완성

      
      > : 태그 내부에 작성(하위 생성)

      + : 태그 다음에 작성(형제)

      *숫자 : 숫자 만큼 생성

      {} : 태그 내부에 작성될 내용
      
      $ : 1 부터 1씩 증가하는 숫자(시퀀스)
